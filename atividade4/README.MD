# 📊 Dashboard de BI (Business Intelligence) - PHP

## INSTRUÇÕES DE USO

* Entre no link desse repositório pelo link https://github.com/Lhgs-pog/POSBaula3
* Clica no botão "Code"
* Use o link mostrado para clonar o seu projeto ou baixe o zip
* Utilizando o terminal, entre na pasta "POSBAULA3"
* Execute o comando "docker compose up -d" para rodar o projeto
* Entre no seu navegador de internet e insira essa url "http://localhost:8080/atividade4/index.php"

## 🚀 Tecnologias Utilizadas
* **Linguagem:** PHP 8.2
* **Servidor Web:** Apache
* **Estilização:** CSS (Flexbox)
* **Infraestrutura:** Docker & Docker Compose
